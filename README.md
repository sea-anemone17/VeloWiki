# VeloWiki

개인 지식 위키 / 공부 기록 아카이브 스타터입니다.

## 기능

현재 기본 뼈대에는 다음 기능이 들어 있습니다.

- 문서 목록
- 새 문서 생성
- 문서 수정/저장
- `localStorage` 자동 저장
- 최근 수정 시각
- `[[문서명]]` 링크
- `[[문서명|보이는 이름]]` 링크
- 없는 문서 클릭 후 생성
- `[[분류:수학]]` 분류
- 분류별 문서 목록
- 하위 문서 자동 표시: `수학/지수함수`
- 상위 문서 자동 표시
- 역링크 자동 표시
- 주석 기본 표시: `[^1]`, `[^1]: 설명`
- 간단 검색
- 랜덤 문서

## 실행 방법

### 1. 간단 실행

`index.html`을 브라우저로 열면 됩니다.

단, 일부 브라우저는 `type="module"` 파일을 로컬 파일로 직접 열 때 막을 수 있습니다.
그럴 때는 아래 방법을 쓰세요.

### 2. 로컬 서버 실행

프로젝트 폴더에서 다음 명령어를 실행합니다.

```bash
python -m http.server 5500
```

그 뒤 브라우저에서 아래 주소를 엽니다.

```txt
http://localhost:5500
```

## 추천 첫 사용법

1. `홈` 문서를 엽니다.
2. `수정` 버튼을 누릅니다.
3. 원하는 문서를 `[[문서명]]`으로 연결합니다.
4. 저장합니다.
5. 링크를 눌러 새 문서를 만듭니다.

## 문법

```md
# 제목

관련 문서: [[수학/지수함수]]
다른 이름으로 표시: [[수학/지수함수|지수함수]]

주석입니다.[^1]

[^1]: 주석 설명입니다.

[[분류:수학]]
[[분류:공부]]
```
